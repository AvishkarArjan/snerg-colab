SNeRG Viewer

Overview

Sparse Neural Radiance Grid (SNeRG) is a method for efficient training and rendering of Neural Radiance Fields (NeRF) by Peter Hedman et at., 2021.

NeRFs are generally computationally very heavy and do not permit the real-time rendering of scenes. SNeRG alleviates that issue by "baking" the NeRF neural network into a sparse voxel representation and a texture map. Instead of the numerous neural network forward pass calls, SNeRG essentially performs a lookup operation, allowing real-time rendering.

Rendering is implemented in WebGL using the THREE.js library, and all examples in the viewer are rendered in real-time by your browser. However, models are large, around 150 MB, so please be patient while they load.

Examples

The buldozer Lego model is from the standard dataset used to evaluate NeRF performance. The rest of the models were created by me.
